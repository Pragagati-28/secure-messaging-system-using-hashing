# 🔐 Secure Messaging System using Hashing & Encryption

## 📌 Project Overview
This project is a Secure Messaging System developed using Node.js, Express, MongoDB, and CryptoJS.  
It ensures secure communication between users by implementing hashing and encryption techniques to protect message integrity and confidentiality.

---

## 🚀 Features

- 👤 User Registration & Login System
- 🔑 Password Authentication with bcrypt hashing
- 💬 Secure One-to-One Messaging
- 🔒 Message Encryption using AES (CryptoJS)
- 🧾 Message Integrity using Hashing (SHA-based)
- 📂 File Sharing Support (Images, Audio, Documents)
- 🕒 Message Timestamp Tracking
- 📊 Login History Tracking (IP & Browser Info)
- 🛡️ Basic Security (Login Attempts & Lock System)

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- CryptoJS
- bcryptjs
- HTML, CSS, JavaScript

---

## 🧠 Security Concepts Implemented

- Hashing for message integrity
- AES encryption for message confidentiality
- Password hashing using bcrypt
- Session-based authentication
- Basic brute-force protection (login attempts tracking)

---

## 📂 Project Structure
models/ → Database Schemas

## 🎯 Project Goal

To demonstrate a secure communication system with real-world cybersecurity principles like encryption, hashing, authentication, and access control.
